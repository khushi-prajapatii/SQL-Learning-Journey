## Dataset

A sample sales dataset was created to practice SQL queries.

Columns:

- order_id
- c_name
- city
- product
- quantity
- price
- order_date

## Practice Scenarios

The queries were written based on business-style questions such as:

- View orders from a specific city
- Find laptop sales
- Review high-value orders
- Sort sales by date
- Retrieve a limited number of records
- Filter data using multiple conditions
