# SQL Group By & Having Clause Practice

## Overview
This project contains SQL practice queries focused on:

- GROUP BY
- Aggregate Functions
- HAVING Clause
- Sorting Results with ORDER BY
- Multi-column Grouping

The dataset consists of movie records containing:

- Movie ID
- Movie Name
- Genre
- Language
- Rating
- Ticket Sales

The dataset contains 20 movie records across multiple genres and languages.

## Practice Scenarios

The queries answer business-style questions such as:

- Number of movies by genre
- Number of movies by language
- Average rating by genre
- Total ticket sales by language
- Highest and lowest rated genres
- Genre-language performance analysis
- Filtering grouped results using HAVING
- Comparing categories using aggregate functions

## Skills Practiced

- Data aggregation
- Grouping records
- Filtering grouped results
- Multi-level grouping
- Business-oriented SQL analysis
- Writing analytical SQL queries
